⟡ Diegetic HTML in 3D

A fully interactive HTML dialogue panel rendered as a texture on a 3D plane, using the new texElementImage2D API.

The <select> and <input> are real DOM elements — keyboard, screen reader and focus all work normally, but they're painted into the 3D scene.

Drag to orbit. Requires Chrome Canary with chrome://flags/#canvas-draw-element.

Feature not supported

This demo requires the experimental html-in-canvas API.

1. Open Chrome Canary
2. Enable chrome://flags/#canvas-draw-element
3. Restart and reload this page.
