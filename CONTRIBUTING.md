# Web Platform Incubator Community Group

This repository is being used for work in the W3C Web Platform Incubator Community Group, governed by the [W3C Community License
Agreement (CLA)](http://www.w3.org/community/about/agreements/cla/). To make substantive contributions,
you must join the CG.

If you are not the sole contributor to a contribution (pull request), please identify all
contributors in the pull request comment.

To add a contributor (other than yourself, that's automatic), mark them one per line as follows:

```
+@github_username
```

If you added a contributor by mistake, you can remove them in a comment with:

```
-@github_username
```

If you are making a pull request on behalf of someone else but you had no part in designing the
feature, you can remove yourself with the above syntax.

## Adding to the awesome-html-in-canvas demos list

Adding a link to a demo is a great way to showcase the community's creativity and help others see what’s possible with HTML-in-Canvas. To contribute to the list:

1. Navigate to the [awesome-html-in-canvas.md](https://github.com/WICG/html-in-canvas/blob/main/awesome-html-in-canvas.md) file.
2. Click the **Edit** (pencil) icon in the top right. This will automatically create a fork of the repository in your account.
3. Add your demo link to the "Pet Projects & Demos " section. 
4. Commit your changes with a descriptive message and links to the hosted demo and the source code included. For example: "Add [project name]\(link-to-demo) to the awesome-html-in-canvas demos list. A brief description of what the demo does. [Source]\(link-to-source)".
5. Submit a [Pull Request](https://help.github.com/articles/using-pull-requests/).
